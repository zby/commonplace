# Scenario Instructions

Use this template to describe a recurring agent task in terms of context loads and hops.

- The opening paragraph should state what the user wants and what the agent must do.
- Each step should name the context needed, the source of that context, the hop count, and whether the cost is fixed or variable.
- Use `Escalation path` only for conditional steps outside the common path.
- Use `Variants` for meaningful differences between environments or workflows that the step table does not capture cleanly.
