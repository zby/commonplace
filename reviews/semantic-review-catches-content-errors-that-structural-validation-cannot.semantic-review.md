- [Grounding — domain coverage] The note claims "each semantic check is a weak oracle with TPR > FPR" and "the four checks are decorrelated (they probe different failure modes), so combining them amplifies signal." The error-correction note it cites develops TPR > FPR and decorrelation as conditions for amplification, but it does so for repeated applications of the same check type (voting across independent samples) or structurally different check types (structural vs. semantic vs. metamorphic). Whether the four semantic checks in this note satisfy decorrelation is asserted, not demonstrated. All four checks use the same mechanism (LLM adversarial reading of a single note), which the error-correction note explicitly warns produces correlated errors: "same model, same prompt, same training data — voting converges to the shared bias." The note extends the error-correction framework to its own checks without addressing whether same-model, same-session semantic checks are actually decorrelated.
