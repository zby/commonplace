WARN:
- [Confidence miscalibration] "Context engineering is not static infrastructure. It improves over deployment time. Teams learn which knowledge to route into context, how to structure it for the model, when to prune accumulated debris, and how to scope what each agent sees." The claim that context engineering improves over deployment time is presented as established fact, but this is the note's own theoretical position (grounded in the deploy-time learning framework, which is itself a proposed framework from this KB). The same pattern appears in the final paragraph: "Deploy-time learning builds the machinery that makes in-context learning effective" asserts a necessary dependency as established rather than proposed.
  Recommendation: Hedge the framework-level claims. E.g., "On this view, deploy-time learning builds the machinery..." or "In the deploy-time learning framework, context engineering improves over deployment time as teams learn..." This preserves the argument while signaling that the three-timescale dependency structure is the note's contribution, not an established result.

- [Anthropomorphic framing] "The learning moved from the model's weights to the system layer that feeds the model's context. That system layer is continuously learning in Herbert Simon's sense: it undergoes permanent changes that improve its capacity for adaptation." Attributing "learning" to the system layer is the note's deliberate theoretical move (invoking Simon's definition), but the phrase "the learning moved" personifies learning as something with agency that relocates itself. The subject performing the relocation is Amodei's analytical move (eliminating weight updates), not the learning itself.
  Recommendation: Reframe to make the agent of relocation explicit: "In doing so he relocated the learning from the model's weights to the system layer..." or "The locus of learning shifted from the model's weights to the system layer..." The Simon attribution in the next sentence is fine — it's a deliberate and sourced theoretical commitment.
