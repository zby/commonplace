# Prose Review

## Purpose

Checks whether the note's framing and prose accurately represent what the note actually establishes.

## Gates

- prose/source-residue
- prose/pseudo-formalism
- prose/confidence-miscalibration
- prose/proportion-mismatch
- prose/orphan-references
- prose/unbridged-cross-domain
- prose/redundant-restatement
- prose/anthropomorphic-framing

## Output format

Report findings as WARN or INFO and include CLEAN coverage for gates that hold.
