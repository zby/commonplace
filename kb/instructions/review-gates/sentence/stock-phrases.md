## Failure mode

A sentence uses a pattern commonly generated by LLMs that adds no content.

## Test

Apply the deletion test to every sentence that matches the patterns below. Report all sentences that fail, not just the first.

**Deletion test: remove the sentence.** If the paragraph reads equally well or better without it, the sentence is filler regardless of its pattern.

Common patterns that warrant the deletion test:

- "This is not just X — it is Y" (the "elevation" move)
- "It's worth noting that..." / "It's important to note that..."
- "In other words, ..." followed by a near-verbatim restatement
- "At its core, ..."
- "The key insight here is that..."

If the deletion test shows the sentence is removable, report as WARN, not INFO. If surrounding text already establishes the point, the stock-phrase sentence is filler even if it "adds something."

## Example (fail)

"This is not just summarization — it is interface design." — the next sentence already says what matters; deleting this loses nothing.
