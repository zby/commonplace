=== FRONTMATTER REVIEW: claw-learning-is-broader-than-retrieval.md ===

Checks applied: 4

CLEAN:
- [Description discrimination] The description "A Claw's learning loop must improve action capacity (classification, planning, communication), not just retrieval --- question-answering is one mode among many" adds mechanism (the parenthetical list of concrete action capacities) and scope (positions retrieval as one mode among many) beyond what the title "Claw learning is broader than retrieval" carries alone. An agent seeing five results about Claw learning would immediately know this note argues for action capacity over retrieval, and which action types it covers.
- [Title composability] "since claw learning is broader than retrieval, the evaluation framework must assess action appropriateness..." reads naturally as a sentence fragment. The claim-form title composes well in link contexts.
- [Claim strength] The claim is genuinely contestable. Someone could argue that all agentic actions reduce to retrieval (the note itself acknowledges this counter-position: "You *can* frame all of these as answering a question") and that a retrieval-oriented learning loop is sufficient. The note takes a specific side in that debate.
- [Title-body alignment] The body delivers exactly what the title promises. It identifies the narrow retrieval framing (section 1), enumerates action modalities beyond retrieval (section 2), details what changes for the learning loop under the broader framing (section 3), and explicitly scopes the relationship to the retrieval frame as "not wrong, just partial" (section 4). No drift in either direction.

Overall: CLEAN
