# Reflective systems

A [reflective system](./definitions/reflective-system.md) contains a **causally connected self-representation**: artifacts that describe the system, that the system's own processes consume, so that editing them changes what it later does. The definition makes the term precise — the declared boundary (which processes, artifacts, and people count as the system), the represented aspects, the causal path — and its companion [actionable theory](./definitions/actionable-theory.md) settles the prior question of when a body of theory can guide an operator's choice of interventions at all. Start with these two.

Reflection is structural, and the middle notes develop what it does not supply. A change loop additionally [requires search, evaluation, and operative retention](./governed-adaptation-requires-search-evaluation-and-retention.md): something must find a candidate change, something must be able to reject it, and the accepted result must be kept where it has force — reflection provides a causal path into that loop, not the loop. [A methodology governs its own extension only as far as it settles the meta-decisions it raises](./a-methodology-governs-its-own-extension-only-as-far-as-it-settles.md): every recommendation a methodology issues raises follow-on decisions — what form the artifact takes, how it is verified, what gives it force — and self-extension stalls at the first one the methodology leaves open. And [reflective coverage is graded across representational forms](./reflective-coverage-is-graded-across-representational-forms.md): a system can freely edit its prose and code while its only reach over the model weights is selecting which sealed model runs.

The design space has occupants. Schmidhuber's Gödel machine — [a proof-governed case of reflective self-modification](./goedel-machines-are-a-proof-governed-case-of-self-modification.md) — internalizes every function of self-change behind a proof gate, at the price of ignoring improvements it cannot prove; its running descendants trade the proof for benchmark evidence. [Commonplace classifies as a reflective system](../reference/commonplace-as-a-reflective-system.md) against this repository's observed history: the same skeleton, with the gate held by tests, validators, and a human maintainer inside the declared boundary. The human supplies what no gate here automates — noticing what to change, judging whether it helped — and the reflective structure is what makes that work accumulate: each accepted improvement lands in the self-representation and compounds.
