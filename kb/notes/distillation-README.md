# distillation

Targeted extraction from a larger body of reasoning into a focused artifact shaped by use case, context budget, or consumer. Orthogonal to [constraining](./constraining-README.md) — you can distil without constraining (extract a skill, still underspecified) or constrain without distilling (store an output, no extraction). A child of [learning-theory](./learning-theory-README.md).

## Notes

- [distillation](./definitions/distillation.md) — the definition: the rhetorical mode shifts to match the target (argumentative → procedural, exploratory → assertive); the dominant mechanism in knowledge work
- [constraining and distillation both trade generality for reliability, speed, and cost](./constraining-and-distillation-both-trade-generality-for-reliability.md) — the two mechanisms compared: same trade, different operation (narrowing vs extracting) and different compound yield
- [evolving understanding needs re-distillation, not composition](./evolving-understanding-needs-re-distillation-not-composition.md) — when the source understanding moves, the distilled artifact must be re-derived; stacking patches composes errors
- [an accepted edit verifies the change, not the rule](./an-accepted-edit-verifies-the-change-not-the-rule.md) — why extracting a reusable rule from an accepted edit is a separate distillation step with a weaker oracle than the edit itself
- [reverse-compression is when LLM output expands without adding information](./reverse-compression-is-when-llm-output-expands-without-adding.md) — the failure mode: expansion without new extractable structure, the inverse of distillation
- [information value is observer-relative](./information-value-is-observer-relative.md) — why distillation creates value at all: deterministic transformations add zero classical information yet make structure accessible to bounded observers
- [minimum viable vocabulary](./minimum-viable-vocabulary-is-the-naming-set-that-most-reduces.md) — the naming set that most reduces extraction cost for a bounded observer; synthesizes information-value, discovery, and distillation
