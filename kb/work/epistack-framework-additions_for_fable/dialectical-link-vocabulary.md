# A dialectical/evidential link vocabulary

Layer: structure.

[ADR 009](../../reference/adr/009-link-relationship-semantics.md) and [ADR 019](../../reference/adr/019-collection-owned-link-vocabulary.md) give collection-owned link semantics; casework wants the argument-mapping set — `supports`, `rebuts`, `undercuts`, `is-evidence-for`, `depends-on` (Toulmin/IBIS-flavored). The LHC case is largely a *dependency chain* of safety arguments; "what does this claim rest on" needs to be a typed edge.

Do not enumerate the full ~15-label ontology up front — grow the vocabulary from need, guarded by the articulation test. The exceptions worth reserving now are `correlated-with` / `independent-of`, the link-level expression of [independence clustering](./independence-clustering.md).
