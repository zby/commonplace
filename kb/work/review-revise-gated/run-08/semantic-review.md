- [completeness-boundary-cases] The three-type trace taxonomy (conversation transcripts, tool/action traces, reasoning traces) does not address hybrid traces where reasoning and tool use are interleaved within a single step. The note's framing ("The conflation arises one layer above the model itself," line 18) implies the problem is architectural, but the clean/problematic boundary is drawn at the interface layer rather than at the behavior. Recommendation: Qualify line 22 to say that raw SDK calls have no *built-in* transcript problem but that application code can recreate the same pattern by manually persisting message arrays.
