# Workshop: Skills vs Instructions

## Question

When should a reusable procedure be a **skill** and when is it enough to add the path to an instruction file in CLAUDE.md?

Both are distilled procedures in the same medium. Both need a name and description always loaded. Both load the body on demand. The agent follows both autonomously. The file format is interchangeable. So what, if anything, actually differs in the loading mechanism — and does that difference matter for when to use which?
