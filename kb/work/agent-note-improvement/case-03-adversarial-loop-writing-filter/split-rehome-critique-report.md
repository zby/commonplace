# Split and Rehome Critique: An adversarial human-agent loop can reconstruct the writing-is-thinking filter

**Target:** `kb/work/agent-note-improvement/case-03-adversarial-loop-writing-filter/baseline-working-tree.md`

## Main note to preserve

**Claim:** A disciplined human-agent loop can relocate the writing-is-thinking filter from solo composition into an adversarial review-and-judgment process, but only while the human remains the judge.

**Minimum argument:**

- Borretti is right that naive AI writing skips the clarifying work of composition and pushes verification to the reader.
- The hidden premise is that concretization and judgment must happen in the same solo act.
- In an agent-operated loop, rendering and judging can be separated: the agent proposes a concretization, gates and critiques pressure it, and the maintainer decides.
- This reconstructs the filter only conditionally: checks must have teeth, and the maintainer must not rubber-stamp.
- The loop does not remove the maintainer's competence floor.

## Rehoming candidates

- **Agent connection work extends one mind's reach over its own corpus** — Agent KB work has a distinct payoff in connection discovery across accumulated notes: the agent can propose relations, contradictions, and extensions that the maintainer can judge but might not retrieve unaided. This needs examples where connection work changed an accepted note or index. It should not remain central in the original note because it answers a different objection than Borretti's single-piece writing-is-thinking critique.

## Deletion candidates

- Delete or rewrite "That labor has no solo equivalent to be contemptible about." It invites an easy objection from ordinary scholarly curation and note systems.
- Delete the implication that the current architecture forces adversarial judgment. Replace it with a weaker claim about making judgment explicit, routed, and reviewable.
- Delete or soften the final "produces better thinking than the solo pen" unless the note explicitly frames it as an empirical bet.

## Revision target

The revised note should stay focused on the reconstructed-filter mechanism: naive delegation loses the stall; an adversarial loop can restore it by separating rendering from judgment; the restoration is conditional and bounded by human competence. Corpus connection work can remain as one compact secondary payoff, but it should not carry its own paragraph unless promoted to a separate note.
