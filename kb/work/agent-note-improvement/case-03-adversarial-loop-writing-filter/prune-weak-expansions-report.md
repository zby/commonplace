# Prune Weak Expansions: An adversarial human-agent loop can reconstruct the writing-is-thinking filter

**Target:** `kb/work/agent-note-improvement/case-03-adversarial-loop-writing-filter/baseline-working-tree.md`
**Strongest retained claim:** A human-agent workflow can preserve the epistemic value of writing only when the adversarial loop, not the initial generation act, becomes the place where vague or weak ideas are concretized and judged.

## Core support

- Opening concession to Borretti: establishes the problem the note must answer rather than dismissing it.
- "Filter lives in one writer's pen" paragraph: carries the main reversal, separating rendering from judging and locating the filter in the loop.
- Conditional paragraph: keeps the claim honest by saying the loop works only if the human remains judge and review stays adversarial.
- Competence-floor paragraph: prevents the note from claiming that agent assistance raises the maintainer beyond their own domain understanding.
- "How this could be wrong": gives the claim a clear failure condition.

## Proposed shape

1. Concede Borretti's writing-is-thinking critique.
2. State the disputed premise: the filter need not live in one writer's first rendering act.
3. Define the loop operation: agent renders; critique and gates pressure the rendering; human accepts, rejects, or revises.
4. State the condition: the loop only works when checks remain adversarial and the human stays judge.
5. Preserve the competence floor.
6. Mention corpus connection work as a secondary payoff, not proof of the writing-filter claim.
7. End with the failure condition.

## Candidate splits

- **Agent connection work extends one mind's reach over its own corpus** — Claim: agent-operated KB work is strongest not when it replaces judgment, but when it searches and proposes connections across a corpus larger than the maintainer can hold in a single pass. Required support: examples from connect reports, review runs, or accepted note links where the agent surfaced a non-obvious relation the maintainer could then judge.

## Net effect

The note becomes harder to attack if it narrows the mechanism from "the architecture forces judgment" to "the architecture makes adversarial judgment available and auditable," and if it demotes corpus connection work from a second thesis to a secondary payoff. The central claim is strong enough; the main edit is pruning overconfident branches around it.
