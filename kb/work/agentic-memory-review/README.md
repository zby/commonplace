# Agentic Memory Review

Workshop for preparing the next agentic memory review pass over the existing related-system landscape.

## Contents

- [systems.csv](./systems.csv) - current repo-backed reviews plus lightweight GitHub-backed coverage with public repositories and local checkout paths.

## Inventory Scope

The CSV lists distinct active systems for review preparation, not every historical review file. At the latest inventory update the review directory had 118 review markdown files excluding `dir-index.md`: 94 non-archived reviews and 24 `.replaced.*` archived reviews. The CSV has 100 rows because it includes the 94 non-archived reviews plus 6 lightweight GitHub-backed systems already visible through source-only or ingest coverage. Source-only entries without a reachable public repository, such as Incremental Self-Improvement, are not listed because the inventory requires a public repo and clone path.

## Closure

Close this workshop when the inventory has been consumed by the review pass and any durable conclusions have been promoted into `kb/agent-memory-systems/` or `kb/notes/`.
