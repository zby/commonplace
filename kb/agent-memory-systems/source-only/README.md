# Source-only agent memory systems

Paper- or article-grounded systems that are useful for the related-systems landscape but do not currently have a repo-backed review in `../reviews/`.

Use this directory for standard `type: note` coverage when the `agent-memory-system-review` type would be wrong. That review type requires a reachable GitHub repository and code inspection.

- [AgeMem](./agemem.md) — RL-trained memory-management policy covered through source ingest and analysis notes.
- [Incremental Self-Improvement](./incremental-self-improvement.md) — Schmidhuber's reward-gated self-modification paradigm for learning learning strategies; useful lineage for policy-level promotion, but source-only and not a modern KB system.
- [Trajectory-Informed Memory Generation](./trajectory-informed-memory-generation.md) — trajectory-to-tip learning pipeline covered through source ingest.
